# Практическая работа №8

## Описание
Задача этой работы состояла в создании приложения с боковым меню `Navigation Drawer` и переходами между несколькими разделами.

## Что было сделано
1. За основу взят шаблон Android Studio с боковой навигацией.
2. Под структуру задания перенастроены пункты меню и граф навигации.
3. Удалены шаблонные экраны, не относящиеся к финальному варианту приложения.
4. Создан общий фрагмент `MenuSectionFragment`, который используется для отображения всех разделов.
5. Для каждого пункта меню через аргументы передаются собственные заголовок и иконка.
6. Строковые ресурсы вынесены в `strings.xml`, а лишние элементы шаблона удалены.
7. Интерфейс каждого раздела сделан единообразным: крупная иконка и подпись по центру экрана.

## Результат
Итогом стало приложение с корректно работающим `Navigation Drawer`, в котором каждый пункт открывает отдельный экран с общей структурой отображения.
